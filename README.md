## vue-shop

### 项目依赖
```
先下载依赖在运行项目
npm install
```

### 运行项目
```
npm run serve
```

### 打包项目
```
npm run build
```

### 项目描述
```
1. 主要页面-src/views/
    1. home页面-home
    2. 登录(login)页面-login
```
```
2. 路由组件-src/views/home|login/*.vue
    1. 登录路由组件-login.vue
    2. home页面组件
        1. 主界面-home.vue
        2. data-项目功能路由
            1. user-用户管理
                1. 用户列表组件-users.vue
            2. power-权限管理
                1. 角色列表组件-roles.vue
                2. 权限列表组件-rights.vue
            3. goods-商品管理
                1. 商品列表组件-goodsList.vue
                2. 添加商品组件-goodsAdd.vue
                3. 分类参数组件-parameter.vue
                4. 商品分类组件-classification.vue
            4. order-订单管理
                1. 订单列表-orders.vue
            5. report-数据统计
                1. 数据报表-reports.vue
```
```
3. 普通组件-src/components
    1. 面包屑组件-crumbs/myBread.vue
```
```
4. 资产
    1. 接口文档
    2. js/city_data2017_element.js-级联选择器省市区版
```
